Fluxo mínimo para demonstrar Login Kit + Content Posting API.

Sandbox demo

Esta página demonstra, de forma mínima, o fluxo de autenticação com TikTok e o envio de um vídeo de teste pela Content Posting API, conforme a documentação oficial.

  • Conectar com TikTok usando OAuth v2 e os scopes solicitados.
  • Trocar o code por um access_token no backend.
  • Disparar o upload de um vídeo de teste para a inbox da conta conectada.
Aguardando ação…

OBS: por padrão, esta página assume que o backend está rodando em http://localhost:8000. Ajuste a constante API_BASE no código desta página para apontar para o domínio em produção quando estiver pronto.
